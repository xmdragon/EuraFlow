# /v2/posting/fbs/digital/act/document-sign

- **HTTP方法**: POST
- **路径**: `/v2/posting/fbs/digital/act/document-sign`

## 详细信息

  * /v2/posting/fbs/act/list — 获取先前生成的操作列表。

  * /v2/posting/fbs/digital/act/document-sign — 如果您连接到电子文档管理(EDI)， 请就差异和盈余签署电子行为签字。
