# 从卖家的系统中改变商品货号

## 接口信息

- **HTTP方法**: `POST`
- **API路径**: `/v1/product/update/offer-id`
- **操作ID**: `operation/ProductAPI_ProductUpdateOfferID`

## 描述

改变附加在商品上的offer_id的方法。您可以改变几个offer_id。

## 请求参数

### Header参数

| Client-Id | required |  | 用户识别号。 |
|---|---|---|---|
| Api-Key | required |  | API-密钥。 |

## 请求示例

```json
{
  "update_offer_id": [
    {
      "new_offer_id": "string",
      "offer_id": "string"
    }
  ]
}
```
