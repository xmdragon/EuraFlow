# 报告信息

## 接口信息

- **HTTP方法**: `POST`
- **API路径**: `/v1/report/info`
- **操作ID**: `operation/ReportAPI_ReportInfo`

## 描述

通过识别码回送有关先前创建的报告的信息。

## 请求参数

### Header参数

| Client-Id | required |  | 用户识别号。 |
|---|---|---|---|
| Api-Key | required |  | API-密钥。 |

## 请求示例

```json
{
  "code": "REPORT_seller_products_924336_1720170405_a9ea2f27-a473-4b13-99f9-d0cfcb5b1a69"
}
```
