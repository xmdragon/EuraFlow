# 将信息标记为已读

## 接口信息

- **HTTP方法**: `POST`
- **API路径**: `/v2/chat/read`
- **操作ID**: `operation/ChatAPI_ChatReadV2`

## 描述

一种将选定的信息和其之前的信息标记为已读的方法。

## 请求参数

### Header参数

| Client-Id | required |  | 用户识别号。 |
|---|---|---|---|
| Api-Key | required |  | API-密钥。 |

## 请求示例

```json
{
  "chat_id": "99feb3fc-a474-469f-95d5-268b470cc607",
  "from_message_id": 3000000000118032000
}
```
