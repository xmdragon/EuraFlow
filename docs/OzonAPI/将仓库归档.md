# 将仓库归档

## 接口信息

- **HTTP方法**: `POST`
- **API路径**: `/v1/warehouse/archive`
- **操作ID**: `operation/ArchiveWarehouseFBS`

## 描述

您可以在 Ozon for dev 开发者社区的讨论区对该方法留下反馈。

## 请求参数

### Header参数

| Client-Idrequired | string用户识别号。 |
|---|---|
| Api-Keyrequired | stringAPI-密钥。 |
