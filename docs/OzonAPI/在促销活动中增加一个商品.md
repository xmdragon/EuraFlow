# 在促销活动中增加一个商品

## 接口信息

- **HTTP方法**: `POST`
- **API路径**: `/v1/actions/products/activate`
- **操作ID**: `operation/PromosProductsActivate`

## 描述

一种向现有促销活动添加商品的方法。

## 请求示例

```json
{
  "action_id": 60564,
  "products": [
    {
      "action_price": 356,
      "product_id": 1389,
      "stock": 10
    }
  ]
}
```
