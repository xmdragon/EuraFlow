# 类别特征列表

## 接口信息

- **HTTP方法**: `POST`
- **API路径**: `/v1/description-category/attribute`
- **操作ID**: `operation/DescriptionCategoryAPI_GetAttributes`

## 描述

获取指定类别和类型的商品特征。

## 请求参数

### Header参数

| Client-Id | required |  | 用户识别号。 |
|---|---|---|---|
| Api-Key | required |  | API-密钥。 |

## 请求示例

```json
{
  "description_category_id": 0,
  "language": "DEFAULT",
  "type_id": 0
}
```
