# 获取用于修改仓库信息的揽收点列表

## 接口信息

- **HTTP方法**: `POST`
- **API路径**: `/v1/warehouse/fbs/update/drop-off/list`
- **操作ID**: `operation/WarehouseAPI_ListDropOffPointsForUpdateFBSWarehouse`

## 描述

```json
{

    "types": [

  },

}
```
