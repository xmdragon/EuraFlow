# 聊天清单

## 接口信息

- **HTTP方法**: `POST`
- **API路径**: ``
- **操作ID**: `operation/ChatAPI_ChatListV2`

## 描述

根据指定的过滤器发回关于聊天的信息。

## 请求参数

### Header参数

| Client-Idrequired | string用户识别号。 |
|---|---|
| Api-Keyrequired | stringAPI-密钥。 |

## 请求示例

```json
{
  "filter": {
    "chat_status": "Opened",
    "unread_only": true
  },
  "limit": 1,
  "offset": 0
}
```
