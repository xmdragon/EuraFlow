# 聊天清单

## 接口信息

- **HTTP方法**: `POSTV3`
- **API路径**: `/v3/chat/list`
- **操作ID**: `operation/ChatAPI_ChatListV3`

## 描述

根据指定的过滤器发回关于聊天的信息。

## 请求参数

### Header参数

| Client-Id | required |  | 用户识别号。 |
|---|---|---|---|
| Api-Key | required |  | API-密钥。 |

## 请求示例

```json
{
  "filter": {
    "chat_status": "Opened",
    "unread_only": true
  },
  "limit": 1,
  "cursor": "304000402034"
}
```
