# 货运取消原因

## 接口信息

- **HTTP方法**: `POST`
- **API路径**: `/v1/posting/fbs/cancel-reason`
- **操作ID**: `operation/PostingAPI_GetPostingFbsCancelReasonV1`

## 描述

返回特定货件的取消原因列表。

## 请求参数

### Header参数

| Client-Idrequired | string用户识别号。 |
|---|---|
| Api-Keyrequired | stringAPI-密钥。 |

## 请求示例

```json
{
  "related_posting_numbers": [
    "73837363-0010-3"
  ]
}
```
