# 获取商品图片

## 接口信息

- **HTTP方法**: `POST`
- **API路径**: `/v2/product/pictures/info`
- **操作ID**: `operation/ProductAPI_ProductInfoPicturesV2`

## 描述

## 请求参数

| Api-Keyrequired | stringAPI-密钥。 |
