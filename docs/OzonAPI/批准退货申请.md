# 批准退货申请

## 接口信息

- **HTTP方法**: `POST`
- **API路径**: `/v2/returns/rfbs/verify`
- **操作ID**: `operation/RFBSReturnsAPI_ReturnsRfbsVerifyV2`

## 描述

该方法允许批准申请并同意接收商品进行检查。

## 请求参数

### Header参数

| Client-Idrequired | string用户识别号。 |
|---|---|
| Api-Keyrequired | stringAPI-密钥。 |

## 请求示例

```json
{
  "return_id": 0,
  "return_method_description": "string"
}
```
