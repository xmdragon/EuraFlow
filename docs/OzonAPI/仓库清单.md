# 仓库清单

## 接口信息

- **HTTP方法**: `POST`
- **API路径**: `/v1/warehouse/list`
- **操作ID**: `operation/WarehouseAPI_WarehouseList`

## 描述

方法返回 FBS 和 rFBS 仓库列表。如需获取 FBO 仓库列表，请使用方法/v1/cluster/list。

## 请求参数

### Header参数

| Client-Id | required |  | 用户识别号。 |
|---|---|---|---|
| Api-Key | required |  | API-密钥。 |
