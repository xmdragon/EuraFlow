# 查询商品添加或更新状态

## 接口信息

- **HTTP方法**: `POST`
- **API路径**: `/v1/product/import/info`
- **操作ID**: `operation/ProductAPI_GetImportProductsInfo`

## 描述

允许获取商品卡片创建或更新的状态。

## 请求参数

### Header参数

| Client-Idrequired | string用户识别号。 |
|---|---|
| Api-Keyrequired | stringAPI-密钥。 |

## 请求示例

```json
{
  "task_id": "172549793"
}
```
