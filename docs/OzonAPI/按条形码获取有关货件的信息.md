# 按条形码获取有关货件的信息

## 接口信息

- **HTTP方法**: `POST`
- **API路径**: `/v2/posting/fbs/get-by-barcode`
- **操作ID**: `operation/PostingAPI_GetFbsPostingByBarcode`

## 描述

用户识别号。

## 请求参数

### Header参数

| Client-Idrequired | string用户识别号。 |
|---|---|
| Api-Keyrequired | stringAPI-密钥。 |

## 请求示例

```json
{
  "barcode": "20325804886000"
}
```
