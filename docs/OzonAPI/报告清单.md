# 报告清单

## 接口信息

- **HTTP方法**: `POST`
- **API路径**: `/v1/report/list`
- **操作ID**: `operation/ReportAPI_ReportList`

## 描述

回送之前已经生成的报告的列表。

## 请求参数

### Header参数

| Client-Idrequired | string用户识别号。 |
|---|---|
| Api-Keyrequired | stringAPI-密钥。 |

## 请求示例

```json
{
  "page": 1,
  "page_size": 1000,
  "report_type": "ALL"
}
```
