# 创建新聊天

## 接口信息

- **HTTP方法**: `POST`
- **API路径**: `/v1/chat/start`
- **操作ID**: `operation/ChatAPI_ChatStart`

## 描述

与买家建立关于的快递的新聊天。例如，为了确定地址或商品型号。

## 请求参数

### Header参数

| Client-Id | required |  | 用户识别号。 |
|---|---|---|---|
| Api-Key | required |  | API-密钥。 |

## 请求示例

```json
{
  "posting_number": "47873153-0052-1"
}
```
