# 可供运输的列表

## 接口信息

- **HTTP方法**: `POST`
- **API路径**: `/v1/posting/carriage-available/list`
- **操作ID**: `operation/PostingAPI_GetCarriageAvailableList`

## 描述

需要打印验收证明书和运输货单的收货方式。

## 请求参数

### Header参数

| Client-Id | required |  | 用户识别号。 |
|---|---|---|---|
| Api-Key | required |  | API-密钥。 |

## 请求示例

```json
{
  "delivery_method_id": 0,
  "departure_date": "2019-08-24T14:15:22Z"
}
```
