# 将快递方式与快递设施联系起来

## 接口信息

- **HTTP方法**: `POST`
- **API路径**: `/v1/polygon/bind`
- **操作ID**: `operation/PolygonAPI_BindPolygon`

## 描述

用户识别号。

## 请求参数

### Header参数

| Client-Id | required |  | 用户识别号。 |
|---|---|---|---|
| Api-Key | required |  | API-密钥。 |

## 请求示例

```json
{
  "delivery_method_id": 0,
  "polygons": [
    {
      "polygon_id": "1323",
      "time": "30"
    }
  ],
  "warehouse_location": {
    "lat": "58.52391272075821",
    "lon": "31.236791610717773"
  }
}
```
