# 确认货件发运日期

## 接口信息

- **HTTP方法**: `POST`
- **API路径**: `/v1/posting/cutoff/set`
- **操作ID**: `operation/PostingAPI_SetPostingCutoff`

## 描述

用于卖家或非集成运输商配送的货件方法。

## 请求参数

### Header参数

| Client-Idrequired | string用户识别号。 |
|---|---|
| Api-Keyrequired | stringAPI-密钥。 |

## 请求示例

```json
{
  "new_cutoff_date": "2019-08-24T14:15:22Z",
  "posting_number": "string"
}
```
