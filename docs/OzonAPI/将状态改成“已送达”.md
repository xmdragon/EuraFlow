# 将状态改成“已送达”

## 接口信息

- **HTTP方法**: `POST`
- **API路径**: `/v2/fbs/posting/delivered`
- **操作ID**: `operation/PostingAPI_FbsPostingDelivered`

## 描述

如果使用第三方快递服务，请将货运状态改成“已送达”。

## 请求参数

### Header参数

| Client-Idrequired | string用户识别号。 |
|---|---|
| Api-Keyrequired | stringAPI-密钥。 |

## 请求示例

```json
{
  "posting_number": [
    "48173252-0033-2"
  ]
}
```
