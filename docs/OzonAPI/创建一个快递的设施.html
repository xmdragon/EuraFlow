创建一个快递的设施

post
/v1/polygon/create
  • 描述和范例
  • 控制台

你可以为快递方式添加一个设施。

通过在https://geojson.io，获取其坐标来创建一个设施：在地图上至少标记3个点，并用线连接起来。

header Parameters
Client-Id
required
string

用户识别号。

Api-Key
required
string

API-密钥。

Request Body schema: application/json
coordinates
required
string

快递设施的坐标，格式为 [[[lat long]]]。

回复

Response Schema: application/json
polygon_id
integer <int64>

设施识别号。

请求范例

    Content type
    application/json
    {
    • "coordinates": "[[[30.149574279785153,59.86550435303646],[30.21205902099609,59.846884387977326],[30.255661010742184,59.86240174913176],[30.149574279785153,59.86550435303646]]]"
    }

    回复范例

    • 200
    • 400
    • 403
    • 404
    • 409
    • 500
    Content type
    application/json
    {
    • "polygonId": "1323"
    }
