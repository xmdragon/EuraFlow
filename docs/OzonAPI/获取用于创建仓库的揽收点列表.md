# 获取用于创建仓库的揽收点列表

## 接口信息

- **HTTP方法**: `POST`
- **API路径**: `/v1/warehouse/fbs/create/drop-off/list`
- **操作ID**: `operation/WarehouseAPI_ListDropOffPointsForCreateFBSWarehouse`

## 描述

您可以在讨论的评论中对此方法提供反馈 在 Ozon for dev 开发者社区中。
