财务报告

post
/v1/finance/cash-flow-statement/list
  • 描述和范例
  • 控制台

从1号到15号以及从16号到31号的财务报告获取方式。 在请求一天的报告时，您将收到15天的报告。 与个人中心中的财务→余额部分相符。

header Parameters
Client-Id
required
string

用户识别号。

Api-Key
required
string

API-密钥。

Request Body schema: application/json
required
object

报告期限。

page
required
integer <int32>

请求返回中的页码。

with_details
boolean

true，如果需要在响应中添加附加参数。

page_size
required
integer <int32>

页面上的元素数量。

回复

Response Schema: application/json
object

方法操作结果。

Array of objects

报告清单。

Array of objects

细节信息。

page_count
integer <int64>

含有报告的页数。

    Content type

      },
    • "with_details": true,
    • "page": 1,
    • "page_size": 1
    }
