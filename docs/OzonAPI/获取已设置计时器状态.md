# 获取已设置计时器状态

## 接口信息

- **HTTP方法**: `POST`
- **API路径**: `/v1/product/action/timer/status`
- **操作ID**: `operation/ProductAPI_ActionTimerStatus`

## 描述

用户识别号。

## 请求参数

### Header参数

| Client-Id | required |  | 用户识别号。 |
|---|---|---|---|
| Api-Key | required |  | API-密钥。 |

## 请求示例

```json
{
  "product_ids": 0
}
```
