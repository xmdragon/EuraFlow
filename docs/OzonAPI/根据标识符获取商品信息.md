# 根据标识符获取商品信息

## 接口信息

- **HTTP方法**: `POST`
- **API路径**: `/v3/product/info/list`
- **操作ID**: `operation/ProductAPI_GetProductInfoList`

## 描述

用于根据商品标识符获取商品信息的方法。

## 请求参数

### Header参数

| Client-Idrequired | string用户识别号。 |
|---|---|
| Api-Keyrequired | stringAPI-密钥。 |

## 请求示例

```json
{
  "offer_id": [
    "string"
  ],
  "product_id": [
    "string"
  ],
  "sku": [
    "string"
  ]
}
```
