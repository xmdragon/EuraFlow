# 将状态改为“由卖家发送”

## 接口信息

- **HTTP方法**: `POST`
- **API路径**: `/v2/fbs/posting/sent-by-seller`
- **操作ID**: `operation/PostingAPI_FbsPostingSentbyseller`

## 描述

将货运状态改为“由卖家发送”。该状态仅适用于从国外销售的头程物流卖家。

## 请求参数

### Header参数

| Client-Id | required |  | 用户识别号。 |
|---|---|---|---|
| Api-Key | required |  | API-密钥。 |

## 请求示例

```json
{
  "posting_number": [
    "47173252-0073-1"
  ]
}
```
