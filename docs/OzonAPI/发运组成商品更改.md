# 发运组成商品更改

## 接口信息

- **HTTP方法**: `POST`
- **API路径**: `/v1/carriage/set-postings`
- **操作ID**: `operation/CarriageAPI_SetPostings`

## 描述

它完全覆盖发运中的订单列表。仅需转交处于等待发运状态下的订单，您就可以将其发运了。

## 请求参数

### Header参数

| Client-Idrequired | string用户识别号。 |
|---|---|
| Api-Keyrequired | stringAPI-密钥。 |

## 请求示例

```json
{
  "carriage_id": 0,
  "posting_numbers": [
    "string"
  ]
}
```
