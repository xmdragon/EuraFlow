# 仓库物流方式清单

## 接口信息

- **HTTP方法**: `POST`
- **API路径**: `/v1/delivery-method/list`
- **操作ID**: `operation/WarehouseAPI_DeliveryMethodList`

## 描述

用户识别号。

## 请求参数

### Header参数

| Client-Idrequired | string用户识别号。 |
|---|---|
| Api-Keyrequired | stringAPI-密钥。 |

## 请求示例

```json
{
  "filter": {
    "provider_id": 424,
    "status": "ACTIVE",
    "warehouse_id": 15588127982000
  },
  "limit": 100,
  "offset": 0
}
```
