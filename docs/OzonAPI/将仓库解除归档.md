# 将仓库解除归档

## 接口信息

- **HTTP方法**: `POST`
- **API路径**: `/v1/warehouse/unarchive`
- **操作ID**: `operation/UnarchiveWarehouseFBS`

## 描述

您可以在 Ozon for dev 开发者社区的讨论区对该方法留下反馈。

## 请求参数

### Header参数

| Client-Id | required |  | 用户识别号。 |
|---|---|---|---|
| Api-Key | required |  | API-密钥。 |

## 请求示例

```json
{
  "warehouse_id": 1020002929332000
}
```
