# 关于商品数量的信息

## 接口信息

- **HTTP方法**: `POST`
- **API路径**: `/v4/product/info/stocks`
- **操作ID**: `operation/ProductAPI_GetProductInfoStocks`

## 描述

返回关于 FBS 和 rFBS 方案下商品数量的信息:

## 请求参数

### Header参数

| Client-Id | required |  | 用户识别号。 |
|---|---|---|---|
| Api-Key | required |  | API-密钥。 |

## 请求示例

```json
{
  "cursor": "",
  "filter": {
    "offer_id": [
      "string"
    ],
    "product_id": [
      "313455276"
    ],
    "visibility": "ALL",
    "with_quant": {
      "created": true,
      "exists": true
    }
  },
  "limit": 100
}
```
