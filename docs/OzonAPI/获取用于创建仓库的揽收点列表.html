获取用于创建仓库的揽收点列表

post
/v1/warehouse/fbs/create/drop-off/list
  • 描述和范例
  • 控制台

您可以在 讨论 的评论中对此方法提供反馈 在 Ozon for dev 开发者社区中。

Request Body schema: application/json
object

坐标。

country_code
required
string

ISO 2格式的国家代码。

is_kgt
required
boolean

true表示商品为超大。

object

搜索参数。

回复

Response Schema: application/json
Array of objects

点位列表。

Array ()
address
string

揽收点地址。

object

揽收点坐标。

discount_percent
number <float>

交付货件的折扣百分比。

id
string

揽收点标识符。

object

为获得发运折扣，需要在此时间前交付货件。

type
string
Enum: "PVZ" "PPZ" "SC"

D揽收点类型：

  • PVZ — 订单取货点，
  • PPZ — 订单接收点，
  • SC — 分拣中心。

      • "address": "москва",
      • "types": [
        • "PPZ"
        ]
      }
    }
