# 获取商品价格信息

## 接口信息

- **HTTP方法**: `POST`
- **API路径**: `/v5/product/info/prices`
- **操作ID**: `operation/ProductAPI_GetProductInfoPrices`

## 描述

用户识别号。

## 请求参数

### Header参数

| Client-Idrequired | string用户识别号。 |
|---|---|
| Api-Keyrequired | stringAPI-密钥。 |

## 请求示例

```json
{
  "cursor": "",
  "filter": {
    "offer_id": [
      "356792"
    ],
    "product_id": [
      "243686911"
    ],
    "visibility": "ALL"
  },
  "limit": 100
}
```
