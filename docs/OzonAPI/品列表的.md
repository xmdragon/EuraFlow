# 品列表的

## 接口信息

- **HTTP方法**: `POST`
- **API路径**: `/v3/product/list`
- **操作ID**: `operation/ProductAPI_GetProductList`

## 描述

用于获取所有商品列表的方法。

## 请求参数

### Header参数

| Client-Idrequired | string用户识别号。 |
|---|---|
| Api-Keyrequired | stringAPI-密钥。 |

## 请求示例

```json
{
  "filter": {
    "offer_id": [
      "136748"
    ],
    "product_id": [
      "223681945"
    ],
    "visibility": "ALL"
  },
  "last_id": "",
  "limit": 100
}
```
