发送文件

post
/v1/chat/send/file
  • 描述和范例
  • 控制台

通过文件的识别码将文件发送到现有的聊天。

header Parameters
Client-Id
required
string

用户识别号。

Api-Key
required
string

API-密钥。

Request Body schema: application/json
base64_content
string

文件为 base64 行形式。

chat_id
required
string

聊天识别码。

name
string

带有扩展名的文件名。

回复

Response Schema: application/json
result
string

请求的处理结果。

请求范例

    Content type
    application/json
    {
    • "base64_content": "iVBORw0KGgoAAAANSUhEUgAAAAEAAAABCAQAAAC1HAwCAAAAC0lEQVR42mNk+A8AAQUBAScY42YAAAAASUVORK5CYII=",
    • "chat_id": "99feb3fc-a474-469f-95d5-268b470cc607",
    • "name": "tempor"
    }
