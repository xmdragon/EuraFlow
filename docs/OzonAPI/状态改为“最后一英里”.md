# 状态改为“最后一英里”

## 接口信息

- **HTTP方法**: `POST`
- **API路径**: `/v2/fbs/posting/last-mile`
- **操作ID**: `operation/PostingAPI_FbsPostingLastMile`

## 描述

如果使用第三方快递服务，请将货运状态改为“最后一英里”。

## 请求参数

### Header参数

| Client-Id | required |  | 用户识别号。 |
|---|---|---|---|
| Api-Key | required |  | API-密钥。 |

## 请求示例

```json
{
  "posting_number": [
    "48173252-0033-2"
  ]
}
```
