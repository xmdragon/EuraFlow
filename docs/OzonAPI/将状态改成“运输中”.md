# 将状态改成“运输中”

## 接口信息

- **HTTP方法**: `POST`
- **API路径**: `/v2/fbs/posting/delivering`
- **操作ID**: `operation/PostingAPI_FbsPostingDelivering`

## 描述

如果使用第三方快递服务，请将货运状态改为“运输中”。

## 请求参数

### Header参数

| Client-Idrequired | string用户识别号。 |
|---|---|
| Api-Keyrequired | stringAPI-密钥。 |

## 请求示例

```json
{
  "posting_number": [
    "33920157-0018-1"
  ]
}
```
