# 货件装运

## 接口信息

- **HTTP方法**: `POST`
- **API路径**: `/v2/posting/fbs/awaiting-delivery`
- **操作ID**: `operation/PostingAPI_MoveFbsPostingToAwaitingDelivery`

## 描述

将有争议的订单转到装运。货件状态将更改为awaiting_deliver。

## 请求参数

### Header参数

| Client-Idrequired | string用户识别号。 |
|---|---|
| Api-Keyrequired | stringAPI-密钥。 |

## 请求示例

```json
{
  "posting_number": [
    "33920143-1195-1"
  ]
}
```
