货件列表（第三版）

post
/v3/posting/fbs/list
  • 描述和范例
  • 控制台

返回指定时间段的货运列表-不应超过一年。

此外，您还可以按货件状态过滤货件。

has_next = true 在响应中表示，不是所有的货物数组都被返回。要获取有关剩余货件的信息，请提出新的含 offset值的请求。

header Parameters
Client-Id
required
string

用户识别号。

Api-Key
required
string

API-密钥。

Request Body schema: application/json
dir
string

分类方向：

  • asc — 从小到大，
  • desc — 从大到小。
required
object

过滤器。

limit
required
integer <int64>

响应中值的数量：

  • 最大值 — 1000,
  • 最小值 — 1。
offset
required
integer <int64>

将在响应中跳过的元素数。 例如，如果“offset=10”，那么响应将从找到的第11个元素开始。

object

要添加到响应的附加字段。

回复

Response Schema: application/json
object

货运数组。

has_next
boolean

响应中未返回整个货运数组的标志:

  • true — 必须提出含其他值 offset的新请求，以获得其他货运信息；
  • false — 响应中返回了在请求中提出的整个用于过滤的货运数组。
Array of objects

货运信息。
