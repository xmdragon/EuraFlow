# 体积重量特征不正确的商品列表

## 接口信息

- **HTTP方法**: `POST`
- **API路径**: `/v1/product/info/wrong-volume`
- **操作ID**: `operation/ProductAPI_ProductInfoWrongVolume`

## 描述

返回体积重量特征不正确的商品列表。如果您已正确填写尺寸，请联系Ozon客服。

## 请求参数

### Header参数

| Client-Id | required |  | 用户识别号。 |
|---|---|---|---|
| Api-Key | required |  | API-密钥。 |

## 请求示例

```json
{
  "cursor": "",
  "limit": 1000
}
```
