每日商品销售报告

post
/v1/finance/realization/by-day
  • 描述和范例
  • 控制台

该方法返回每日商品销售报告中的销售金额数据。不包括取消和无人认领的订单。数据仅可获取从当前日期起最多32个自然日之内的记录。此方法仅对Premium Plus订阅的用户开放。

required
string

年。
