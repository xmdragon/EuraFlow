创建仓库

post
/v1/warehouse/fbs/create
  • 描述和范例
  • 控制台

如果创建带有揽收点配送的仓库，请使用方法/v1/warehouse/fbs/create/drop-off/list获取点位列表。

您可以在 讨论 的评论中对此方法提供反馈 在 Ozon for dev 开发者社区中。

Request Body schema: application/json
required
object

仓库地址坐标。

drop_off_point_id
integer <int64>

揽收点标识符。

first_mile_type
required
string
Enum: "PICK_UP" "DROP_OFF"

头程物流类型：

  • PICK_UP — 订单交给快递员；
  • DROP_OFF — 订单送到接收点。
is_kgt
required
boolean

true表示商品为超大。

name
required
string

仓库名称。

object

仓库参数。

phone
required
string

仓库联系电话。请按格式填写：+7 (XXX) XXX-XX-XX。

working_days
Array of strings <= 7 items
Items Enum: "MONDAY" "TUESDAY" "WEDNESDAY" "THURSDAY" "FRIDAY" "SATURDAY" "SUNDAY"

仓库工作日：

  • MONDAY — 星期一，
  • TUESDAY — 星期二，
  • WEDNESDAY — 星期三，
  • THURSDAY — 星期四，
  • FRIDAY — 星期五，
  • SATURDAY — 星期六，
  • SUNDAY — 星期天。

回复

Response Schema: application/json
operation_id
string

创建FBS仓库的操作标识符。要获取操作状态，请使用方法 /v1/warehouse/operation/status。
