# 仓库列表

## 接口信息

- **HTTP方法**: `POST`
- **API路径**: `/v2/warehouse/list`
- **操作ID**: `operation/WarehouseListV2`

## 描述

该方式返回 FBS 和 rFBS 仓库列表。如要获取 FBO 仓库列表，请使用/v1/warehouse/fbo/list方式。

## 请求参数

### Header参数

| Client-Idrequired | string用户识别号。 |
|---|---|
| Api-Keyrequired | stringAPI-密钥。 |

## 请求示例

```json
{
  "warehouse_ids": [
    20605650762000
  ]
}
```
