# 获取商品详细信息

## 接口信息

- **HTTP方法**: `POST`
- **API路径**: `/v1/product/info/description`
- **操作ID**: `operation/ProductAPI_GetProductInfoDescription`

## 描述

用户识别号。

## 请求参数

### Header参数

| Client-Idrequired | string用户识别号。 |
|---|---|
| Api-Keyrequired | stringAPI-密钥。 |

## 请求示例

```json
{
  "offer_id": "5",
  "product_id": 73453843
}
```
