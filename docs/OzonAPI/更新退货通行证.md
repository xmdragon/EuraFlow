# 更新退货通行证

## 接口信息

- **HTTP方法**: `POST`
- **API路径**: `/v1/return/pass/update`
- **操作ID**: `operation/returnPassUpdate`

## 描述

用户识别号。

## 请求参数

### Header参数

| Client-Id | required |  | 用户识别号。 |
|---|---|---|---|
| Api-Key | required |  | API-密钥。 |

## 请求示例

```json
{
  "arrival_passes": [
    {
      "arrival_pass_id": 0,
      "arrival_time": "2019-08-24T14:15:22Z",
      "driver_name": "string",
      "driver_phone": "string",
      "vehicle_license_plate": "string",
      "vehicle_model": "string"
    }
  ]
}
```
