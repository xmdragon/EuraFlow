# 更新仓库

## 接口信息

- **HTTP方法**: `POST`
- **API路径**: `/v1/warehouse/fbs/update`
- **操作ID**: `operation/UpdateWarehouseFBS`

## 描述

您可以在讨论的评论中对此方法提供反馈 在 Ozon for dev 开发者社区中。

## 请求参数

### Header参数

| Client-Id | required |  | 用户识别号。 |
|---|---|---|---|
| Api-Key | required |  | API-密钥。 |
