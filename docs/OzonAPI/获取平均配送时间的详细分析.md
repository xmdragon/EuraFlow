# 获取平均配送时间的详细分析

## 接口信息

- **HTTP方法**: `POST`
- **API路径**: `/v1/analytics/average-delivery-time/details`
- **操作ID**: `operation/AnalyticsAPI_AverageDeliveryTimeDetails`

## 描述

本方法对应卖家个人中心的分析 → 配送范围 → 平均配送时间模块。

## 请求示例

```json
{
  "cluster_id": 154,
  "filters": {
    "delivery_schema": "ALL",
    "supply_period": "EIGHT_WEEKS"
  },
  "limit": 1000,
  "offset": 0
}
```
