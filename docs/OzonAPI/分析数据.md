# 分析数据

## 接口信息

- **HTTP方法**: `POST`
- **API路径**: `/v1/analytics/data`
- **操作ID**: `operation/AnalyticsAPI_AnalyticsGetData`

## 描述

适用于订阅了Premium Plus的卖家。

## 请求参数

### Header参数

| Client-Id | required |  | 用户识别号。 |
|---|---|---|---|
| Api-Key | required |  | API-密钥。 |

## 请求示例

```json
{
  "date_from": "2020-09-01",
  "date_to": "2021-10-15",
  "metrics": [
    "hits_view_search"
  ],
  "dimension": [
    "sku",
    "day"
  ],
  "filters": [],
  "sort": [
    {
      "key": "hits_view_search",
      "order": "DESC"
    }
  ],
  "limit": 1000,
  "offset": 0
}
```
