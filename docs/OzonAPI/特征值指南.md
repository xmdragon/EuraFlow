# 特征值指南

## 接口信息

- **HTTP方法**: `POST`
- **API路径**: `/v1/description-category/attribute/values`
- **操作ID**: `operation/DescriptionCategoryAPI_GetAttributeValues`

## 描述

返回特征值指南。

## 请求参数

### Header参数

| Client-Id | required |  | 用户识别号。 |
|---|---|---|---|
| Api-Key | required |  | API-密钥。 |

## 请求示例

```json
{
  "attribute_id": 0,
  "description_category_id": 0,
  "language": "DEFAULT",
  "last_value_id": 0,
  "limit": 0,
  "type_id": 0
}
```
