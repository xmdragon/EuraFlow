# 更新库存商品的数量

## 接口信息

- **HTTP方法**: `POST`
- **API路径**: `/v2/products/stocks`
- **操作ID**: `operation/ProductAPI_ProductsStocksV2`

## 描述

可以改变一个商品的库存数量信息。

## 请求参数

### Header参数

| Client-Id | required |  | 用户识别号。 |
|---|---|---|---|
| Api-Key | required |  | API-密钥。 |

## 请求示例

```json
{
  "stocks": [
    {
      "offer_id": "PH11042",
      "product_id": 313455276,
      "stock": 100,
      "warehouse_id": 22142605386000
    }
  ]
}
```
