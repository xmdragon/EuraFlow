# 添加商品产地信息

## 接口信息

- **HTTP方法**: `POST`
- **API路径**: `/v2/posting/fbs/product/country/set`
- **操作ID**: `operation/PostingAPI_SetCountryProductFbsPostingV2`

## 描述

将“产地”商品属性添加到方法中，如果该信息未指定。

## 请求示例

```json
{
  "country_iso_code": "NO",
  "posting_number": "57195475-0050-3",
  "product_id": 180550365
}
```
