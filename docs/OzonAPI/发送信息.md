# 发送信息

## 接口信息

- **HTTP方法**: `POST`
- **API路径**: `/v1/chat/send/message`
- **操作ID**: `operation/ChatAPI_ChatSendMessage`

## 描述

通过识别码向现有聊天发送消息。

## 请求参数

### Header参数

| Client-Idrequired | string用户识别号。 |
|---|---|
| Api-Keyrequired | stringAPI-密钥。 |

## 请求示例

```json
{
  "chat_id": "99feb3fc-a474-469f-95d5-268b470cc607",
  "text": "test"
}
```
