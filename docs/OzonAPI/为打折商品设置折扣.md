# 为打折商品设置折扣

## 接口信息

- **HTTP方法**: `POST`
- **API路径**: `/v1/product/update/discount`
- **操作ID**: `operation/ProductAPI_ProductUpdateDiscount`

## 描述

FBS模式下出售折扣商品的折扣幅度设置方法。

## 请求参数

### Header参数

| Client-Idrequired | string用户识别号。 |
|---|---|
| Api-Keyrequired | stringAPI-密钥。 |

## 请求示例

```json
{
  "discount": 0,
  "product_id": 0
}
```
