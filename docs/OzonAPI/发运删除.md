# 发运删除

## 接口信息

- **HTTP方法**: `POST`
- **API路径**: `/v1/carriage/cancel`
- **操作ID**: `operation/CarriageAPI_CarriageCancel`

## 描述

用户识别号。

## 请求参数

### Header参数

| Client-Id | required |  | 用户识别号。 |
|---|---|---|---|
| Api-Key | required |  | API-密钥。 |

## 请求示例

```json
{
  "carriage_id": 0
}
```
