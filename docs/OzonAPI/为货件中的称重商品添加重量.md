# 为货件中的称重商品添加重量

## 接口信息

- **HTTP方法**: `POST`
- **API路径**: `/v2/posting/fbs/product/change`
- **操作ID**: `operation/PostingAPI_ChangeFbsPostingProduct`

## 描述

用户识别号。

## 请求参数

### Header参数

| Client-Id | required |  | 用户识别号。 |
|---|---|---|---|
| Api-Key | required |  | API-密钥。 |

## 请求示例

```json
{
  "items": [
    {
      "sku": 1231428352,
      "weightReal": [
        0.3
      ]
    }
  ],
  "posting_number": "33920158-0006-1"
}
```
