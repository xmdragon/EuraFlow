搜集订单 (第4方案)

post
/v4/posting/fbs/ship
  • 描述和范例
  • 控制台

拆分订单，并将状态改为awaiting_deliver。

packages中的每个元素都可以包含多个products和货物。 products中的每个元素是包含在这批货物中的商品。

如果出现以下情况，需要拆分订单：

  • 商品在一个包装里放不下，
  • 商品不可以放在一个包装里。

如需拆分订单，请在packages数组中传递多个对象。

不需要拆分订单的请求示例：两个商品将在一个货件中发货。

{
  "packages": [
    {
      "products": [
        {
          "product_id": 185479045,
          "quantity": 2
        }
      ]
    }
  ],
  "posting_number": "89491381-0072-1"
}

需要拆分订单的请求示例：每个商品将在单独的货件中发货。

{
  "packages": [
    {
      "products": [
        {
          "product_id": 185479045,
          "quantity": 1
        }
      ]
    },
    {
      "products": [
        {
          "product_id": 185479045,
          "quantity": 1
        }
      ]
    }
  ],
  "posting_number": "89491381-0072-1"
}

请使用/v5/fbs/posting/product/exemplar/set方式输入信息。
