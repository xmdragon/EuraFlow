# 商品报告

## 接口信息

- **HTTP方法**: `POST`
- **API路径**: `/v1/report/products/create`
- **操作ID**: `operation/ReportAPI_CreateCompanyProductsReport`

## 描述

获得带有商品数据的报告的方法。例如，Ozon的ID，商品的数量，价格，状态。
与个人中心中的商品和价格→商品列表→下载商品CSV部分相符。

## 请求参数

### Header参数

| Client-Id | required |  | 用户识别号。 |
|---|---|---|---|
| Api-Key | required |  | API-密钥。 |

## 请求示例

```json
{
  "language": "DEFAULT",
  "offer_id": [],
  "search": "",
  "sku": [],
  "visibility": "ALL"
}
```
