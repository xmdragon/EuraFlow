# 聊天历史记录

## 接口信息

- **HTTP方法**: `POST`
- **API路径**: ``
- **操作ID**: `operation/ChatAPI_ChatHistoryV2`

## 描述

恢复聊天室消息历史记录。默认顺序为从最新消息到之前的消息。

## 请求参数

### Header参数

| Client-Id | required |  | 用户识别号。 |
|---|---|---|---|
| Api-Key | required |  | API-密钥。 |

## 请求示例

```json
{
  "chat_id": "18b8e1f9-4ae7-461c-84ea-8e1f54d1a45e",
  "direction": "Forward",
  "from_message_id": 3000000000118032000,
  "limit": 1
}
```
