# 将订单拆分为不带备货的货件

## 接口信息

- **HTTP方法**: `POST`
- **API路径**: `/v1/posting/fbs/split`
- **操作ID**: `operation/FbsSplit`

## 描述

您可以在讨论的评论中对此方法提供反馈 在 Ozon for dev 开发者社区中。

## 请求示例

```json
{
  "posting_number": "string",
  "postings": [
    {
      "products": [
        {
          "product_id": 0,
          "quantity": 0
        }
      ]
    }
  ]
}
```
